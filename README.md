> **The Industrial-Grade AI Career Engine.** Don't just apply. Maximize your potential with AI-driven resume tailoring and automated job tracking.

## ✨ Key Features

### 🤖 AI-Powered Tailoring
Stop sending generic resumes. Our OpenAI-integrated engine analyzes job descriptions and transforms your experience into a high-impact, ATS-optimized masterpiece.

### 📊 Professional Dashboard
A command center for your career. Track applications, manage versions, and watch your activity telemetry grow as you push toward your next role.

### 🛡️ The Resume Vault
Never lose a great version again. Every generation is safely stored in our secure vault, allowing you to compare, snapshot, and deploy the perfect version for any role.

### 🛰️ Live Activity Feed
Real-time telemetry of your career progress. Monitor your application velocity and see your growth in real-time.

## 🛠️ Tech Stack

- **Frontend:** [Next.js 15](https://nextjs.org/) + [TypeScript](https://www.typescriptlang.org/) + [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
- **Backend:** [FastAPI](https://fastapi.tiangolo.com/) (Python 3.12+)
- **Auth:** [Clerk](https://clerk.com/) (Enterprise-grade Identity)
- **Database:** [MySQL](https://www.mysql.com/) + [SQLAlchemy](https://www.sqlalchemy.org/) + [Alembic](https://alembic.sqlalchemy.org/)
- **AI Engine:** [OpenAI GPT-4o](https://openai.com/)
- **Infrastructure:** [Railway](https://railway.app/) (Backend) + [Vercel](https://vercel.com/) (Frontend)

## 🚀 Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/daikendy/ResumeMaxxing.git
cd ResumeMaxxing
```

### 2. Backend Setup
```bash
cd backend
python -m venv venv
./venv/Scripts/activate
pip install -r requirements.txt
alembic upgrade head
python main.py
```

### 3. Frontend Setup
```bash
cd ../frontend
npm install
npm run dev
```
